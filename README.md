# 🎯 FocusRoom — Study Together, Live

Real-time collaborative study room. Open the link, share the code, study together on a live canvas.

> ✅ Firebase is already connected. Just follow the 2 steps below to go live.

---

## STEP 1 — Upload to GitHub

1. Go to **https://github.com** → Sign up (free) → Log in
2. Click the **"+"** icon (top right) → **"New repository"**
3. Name it `focusroom` → Keep it **Public** → Click **"Create repository"**
4. On the next page, click **"uploading an existing file"**
5. Drag and drop ALL files from this ZIP
6. Click **"Commit changes"**

---

## STEP 2 — Deploy on Vercel

1. Go to **https://vercel.com** → **"Sign up"** → **"Continue with GitHub"**
2. Click **"Add New Project"** → Import `focusroom`
3. Leave all settings default → Click **"Deploy"**
4. Done! You get a live link like: **`https://focusroom-xyz.vercel.app`** 🎉

Share that link — friends enter the same room code and you study together live!
